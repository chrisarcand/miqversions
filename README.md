### Installation

#### Homebrew (MacOS)

```bash
$ brew tap chrisarcand/formulae
$ brew install miqversions
```

#### Other platforms

[Golang](https://golang.org/) must be installed.

```bash
$ go get github.com/chrisarcand/miqversions
```
